OnslaughTFC Post root linux attack kit
======================================


How to run:
==========


first get onslaught.c
---------------------
    wget raw.github.com/TheFailCollective/OnslaughTFC/master/onslaugt.c

then compile
-------------
    gcc onslaught.c -o onslaught
    chmod +x onslaught

then run
--------
    ./onslaught

type help for a list of commands.
